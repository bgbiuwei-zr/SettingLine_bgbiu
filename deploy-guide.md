# 国内部署指南

## 腾讯云静态网站托管

1. 访问：https://console.cloud.tencent.com/tcb/env/index
2. 创建环境 → 选择按量付费
3. 左侧菜单 → 云托管 → 静态网站
4. 上传 dist 文件夹内容
5. 获得访问域名

## 阿里云OSS

1. 访问：https://oss.console.aliyun.com
2. 创建存储桶 → 公共读
3. 上传 dist 文件夹内容  
4. 开启静态网站托管
5. 获得访问域名

## 七牛云静态网站

1. 访问：https://portal.qiniu.com
2. 对象存储 → 创建空间
3. 上传文件
4. 绑定域名

所有平台都有免费额度，足够个人使用。
